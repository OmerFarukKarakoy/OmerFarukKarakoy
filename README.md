- **📊 [Project B: Crypto Trading Strategy](https://github.com/OmerFarukKarakoy/Crypto-Trading-Strategy)**

- **📱 [Project D: DailyEnglish](https://github.com/OmerFarukKarakoy/DailyEnglish)**
